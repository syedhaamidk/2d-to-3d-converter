## üöÄ How to Use

### CLI (Command Line)
```bash
# Braille
python advanced_converter.py braille "Your Text" output.stl

# QR Code
python advanced_converter.py qr "https://example.com" qr.stl --stamp

# Topographic Map
python advanced_converter.py topo --demo terrain.stl

# AI Depth
python advanced_converter.py depth photo.jpg depth.stl

# Multi-Material
python advanced_converter.py multi logo.png dual_color

# Basic Height Map
python advanced_converter.py heightmap image.jpg model.stl
```

### Web Interface
1. Open `web_interface.html` in browser
2. Select mode from 6 options
3. Upload/enter data
4. Customize settings
5. Convert and download!
